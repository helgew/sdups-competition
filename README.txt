A wordpress plugin that generates voting screens and tabulates results for the SDUPS monthly competition.

== Description ==

This is a plugin that will generate voting screens and result tabulation screens for the SDUPS
monthly image and video competition. It is not intended for general use, but if you see anything
useful in the code, feel free to copy and modify for your own needs or plugins.

== Installation ==

1. Upload `sdups-competition.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('sdups_competition_hook'); ?>` in your templates

== Frequently Asked Questions ==

= Can I use this plugin for my own site? =

Absolutely! It will probably not destroy your site and incinerate your servers. Probably!

= What would I use this plugin for? =

No idea. Unless you are running the SDUPS website, it is probably of no use whatsoever.
